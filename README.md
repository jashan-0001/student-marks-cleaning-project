# Student Marks Data Cleaning Project

## Project Overview

This project demonstrates how to clean a student marks dataset using Python and Pandas.

## Dataset

The dataset contains:

- Student Name
- Class
- Maths Marks
- Science Marks
- English Marks
- Attendance

## Cleaning Steps

- Removed duplicate records
- Removed extra spaces
- Converted marks into numeric values
- Filled missing values
- Removed invalid marks
- Reset index
- Saved cleaned dataset
